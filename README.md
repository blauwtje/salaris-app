# Salaris & Urenregistratie

Lokale web-app voor urenregistratie en salarisberekening (NL 2026-loonheffing).

**Openen:** https://blauwtje.github.io/salaris-app/

Alles wordt lokaal in je eigen browser bewaard (`localStorage`); er gaat geen data
naar een server. Elk profiel houdt zijn eigen uren bij, per apparaat gescheiden.
